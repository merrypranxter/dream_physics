## Quick Reference: Best Starting Prompts

**For surreal landscapes:**
`non-euclidean dream architecture, impossible staircase, rooms that shouldn't fit, MC Escher style`

**For atmospheric mood:**
`dream physics palette: deep indigo, phosphene green, memory amber, void black, lucid silver`

**For identity/character:**
`identity shifting in dream, person morphing into multiple selves simultaneously, fluid identity`

**For physics/concepts:**
`emotional field theory, love bending dreamscape toward reunion, warm geometric distortion`

## Prompt Construction Tips

All prompts can be enhanced by appending:
- **Style**: `oil painting`, `digital art`, `photography`, `pencil sketch`, `watercolor`
- **Artist**: `in the style of Salvador Dali`, `Zdzisław Beksiński`, `HR Giger`, `Remedios Varo`
- **Quality**: `highly detailed`, `4K`, `masterpiece`, `award winning`
- **Lighting**: `bioluminescent`, `golden hour`, `void black lighting`, `phosphorescent`
